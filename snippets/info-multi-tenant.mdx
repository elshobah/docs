<Info>
**Isolasi Data Per Lembaga**

Setiap lembaga di Mazid memiliki data yang sepenuhnya terisolasi. Admin Lembaga A hanya bisa melihat dan mengelola data santri/siswa, keuangan, dan pengguna di Lembaga A.

Sistem keamanan Mazid menggunakan enkripsi database tingkat enterprise untuk memastikan data Anda tetap privat dan aman.
</Info>
